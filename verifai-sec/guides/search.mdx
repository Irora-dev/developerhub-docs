title: Search guide
description: How to search SEC filings with VerifAI SEC
---

VerifAI SEC provides search across 20M+ filings. The search endpoint performs case-insensitive matching against company name, ticker, form type, and filing summary.

## Basic search

Pass a search term via the `q` query parameter:

```bash
curl -H "Authorization: Bearer YOUR_API_KEY" \
  "https://api.irora.ai/verifai-sec/v1/filings/search?q=apple"
```

The search matches against multiple fields simultaneously:

| Field     | Example matches                            |
| --------- | ------------------------------------------ |
| `company` | "Apple Inc.", "Pineapple Energy"           |
| `ticker`  | "AAPL"                                     |
| `form`    | "10-K", "8-K", "DEF 14A"                  |
| `summary` | Any text in the AI-generated summary       |

## Response format

```json
{
  "query": "apple",
  "total": 2,
  "results": [
    {
      "company": "Apple Inc.",
      "ticker": "AAPL",
      "cik": "0000320193",
      "form": "10-K",
      "filed": "2026-01-31",
      "summary": "Annual report. Revenue $412B (+8.2% YoY). Services segment $96B. 21 extractable sections."
    }
  ]
}
```

## Search tips

### Search by ticker

Ticker symbol search is the fastest way to find a specific company:

```bash
curl -H "Authorization: Bearer YOUR_API_KEY" \
  "https://api.irora.ai/verifai-sec/v1/filings/search?q=TSLA"
```

### Search by form type

Find all filings of a specific type:

```bash
curl -H "Authorization: Bearer YOUR_API_KEY" \
  "https://api.irora.ai/verifai-sec/v1/filings/search?q=13F"
```

### Search by content

The summary field is included in search, so you can find filings by topic:

```bash
curl -H "Authorization: Bearer YOUR_API_KEY" \
  "https://api.irora.ai/verifai-sec/v1/filings/search?q=restatement"
```

## Full-text search (coming soon)

The Full-Text Search API (`POST /api/full-text-search`) will support advanced query syntax including:

- **Boolean operators**: `revenue AND decline`
- **Phrase matching**: `"material weakness"`
- **Wildcards**: `restatemnt*`
- **Field-specific search**: `company:apple AND form:10-K`
- **Date range filters**: `filed:[2025-01-01 TO 2025-12-31]`

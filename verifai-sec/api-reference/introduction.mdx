The VerifAI SEC API provides programmatic access to SEC filing data, search, and AI-powered analysis.

## Base URL

```
https://api.irora.ai/verifai-sec/v1
```

For local development:

```
http://localhost:3456/api
```

## Authentication

All endpoints (except `/health` and `/pricing`) require a Bearer token:

```bash
curl -H "Authorization: Bearer YOUR_API_KEY" \
  https://api.irora.ai/verifai-sec/v1/filings/recent
```

Get your API key at [irora.ai/keys](https://irora.ai/keys).

## Endpoints

The interactive playground below lets you test every endpoint directly from the docs. Click any endpoint to expand it, fill in parameters, and execute requests.

### System

| Method | Path      | Description           |
| ------ | --------- | --------------------- |
| GET    | `/health` | Service health check  |
| GET    | `/stats`  | Platform statistics   |

### Filings

| Method | Path               | Description                  |
| ------ | ------------------ | ---------------------------- |
| GET    | `/filings/recent`  | Recent filings with metadata |
| GET    | `/filings/search`  | Search filings by query      |

### Reference

| Method | Path        | Description           |
| ------ | ----------- | --------------------- |
| GET    | `/forms`    | Form type categories  |
| GET    | `/products` | API product catalog   |
| GET    | `/pricing`  | Pricing and rate limits |

## Response format

All responses return JSON. See [standard response format](/platform/errors) for the error envelope structure.

## Rate limits

Rate limits depend on your subscription tier. See [rate limits](/platform/rate-limits) for details per endpoint and tier.

<Tip>
  The free tier includes **1,000 API calls/month** — no credit card required.
</Tip>

<Steps>
  <Step title="Get your API key" icon="key">
    Sign up at [irora.ai/keys](https://irora.ai/keys) and create an Irora SEC API key.
  </Step>

  <Step title="Check the service" icon="heart-pulse">
    Verify connectivity with the health endpoint (no auth required):

    ```bash
    curl https://api.irora.ai/irora-sec/v1/health
    ```

    ```json
    {
      "status": "ok",
      "service": "irora-sec",
      "timestamp": "2026-03-03T12:00:00.000Z"
    }
    ```
  </Step>

  <Step title="Fetch recent filings" icon="file-lines">
    Get the latest SEC filings with AI-generated summaries and risk scores:

    <CodeGroup>
    ```bash curl
    curl -H "Authorization: Bearer YOUR_API_KEY" \
      https://api.irora.ai/irora-sec/v1/filings/recent
    ```

    ```javascript Node.js
    const res = await fetch(
      "https://api.irora.ai/irora-sec/v1/filings/recent",
      { headers: { Authorization: "Bearer YOUR_API_KEY" } }
    );
    const data = await res.json();
    console.log(data.filings);
    ```

    ```python Python
    import requests

    res = requests.get(
        "https://api.irora.ai/irora-sec/v1/filings/recent",
        headers={"Authorization": "Bearer YOUR_API_KEY"},
    )
    print(res.json()["filings"])
    ```
    </CodeGroup>

    Each filing includes:

    | Field | Description |
    |-------|-------------|
    | `company` | Company name |
    | `ticker` | Stock ticker symbol |
    | `cik` | SEC Central Index Key |
    | `formType` | Form type (10-K, 8-K, etc.) |
    | `status` | `verified`, `flagged`, or `pending` |
    | `riskScore` | AI risk score from 0 (safe) to 100 (risk) |
    | `summary` | AI-generated filing summary |
  </Step>

  <Step title="Search filings" icon="magnifying-glass">
    Search by company name, ticker, or form type:

    ```bash
    curl -H "Authorization: Bearer YOUR_API_KEY" \
      "https://api.irora.ai/irora-sec/v1/search?query=tesla&formType=10-K"
    ```

    ```json
    {
      "data": [
        {
          "company": "Tesla Inc.",
          "ticker": "TSLA",
          "formType": "10-K",
          "filedDate": "2026-02-14",
          "snippet": "Annual report. Revenue $112B. Automotive margin 18.2%.",
          "score": 0.95
        }
      ],
      "total": 1,
      "page": 1,
      "pageSize": 20,
      "hasMore": false
    }
    ```
  </Step>

  <Step title="Explore the platform" icon="compass">
    Dive deeper into the full suite of Irora SEC capabilities.
  </Step>
</Steps>

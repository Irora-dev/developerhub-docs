The Irora SEC API provides programmatic access to SEC filings, insider trades, politician trades, change detection, watchlists, alerts, and AI-powered analysis.

## Base URL

```
https://api.irora.ai/irora-sec/v1
```

For local development:

```
http://localhost:3456/api
```

## Authentication

All endpoints (except `/health` and `/pricing`) require a Bearer token:

```bash
curl -H "Authorization: Bearer YOUR_API_KEY" \
  https://api.irora.ai/irora-sec/v1/filings/recent
```

<Note>
  Get your API key at [irora.ai/keys](https://irora.ai/keys). The free tier includes 1,000 calls/month.
</Note>

## Pagination

All paginated endpoints use a standard response envelope:

```json
{
  "data": [...],
  "total": 142,
  "page": 1,
  "pageSize": 20,
  "hasMore": true
}
```

Query parameters: `page` (1-indexed, default 1), `pageSize` (default 20, max 100), `sortBy`, `sortDir` (`asc` or `desc`).

## Date ranges

Many endpoints accept a `dateRange` parameter as a shortcut:

| Value | Meaning |
|-------|---------|
| `1D` | Last 24 hours |
| `7D` | Last 7 days |
| `30D` | Last 30 days |
| `90D` | Last 90 days |
| `1Y` | Last year |
| `ALL` | All time |

Alternatively, use `dateFrom` and `dateTo` for custom ranges (ISO 8601 dates).

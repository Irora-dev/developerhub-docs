Irora SEC offers three subscription tiers. All tiers include access to the full EDGAR archive (20M+ filings, 30+ years). Higher tiers unlock more API calls, AI summaries, watchlists, alert rules, and programmatic API key access.

| API calls/month | 1,000 | 50,000 | 200,000 |
| AI summaries/month | 5 | 500 | 2,000 |

| Alert rules | 3 | 50 | Unlimited |
| API keys | 0 | 5 | 25 |

## Rate limits

Rate limits apply per API key and are enforced on a per-second basis. When you exceed the limit, the API returns `429 Too Many Requests` with a `Retry-After` header.

See [rate limits](/platform/rate-limits) for per-endpoint limits by tier.

## Usage tracking

Your current usage is available via the dashboard or the API. Each usage record includes:

| Field | Type | Description |
|-------|------|-------------|
| `metric` | string | `api_calls`, `ai_summaries`, `exports`, or `searches` |
| `count` | integer | Current usage count |
| `limit` | integer | Tier limit for this metric |
| `periodStart` | string | Billing period start (ISO 8601) |
| `periodEnd` | string | Billing period end (ISO 8601) |

## Upgrading

Upgrade your tier at any time from the billing settings in the dashboard. Changes take effect immediately and are prorated for the current billing period.

You can also retrieve pricing information programmatically:

```bash
# No auth required
curl "https://api.irora.ai/irora-sec/v1/pricing"
```

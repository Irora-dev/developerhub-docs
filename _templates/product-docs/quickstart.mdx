## 1. Get your API key

Sign up at [irora.ai/keys](https://irora.ai/keys) and create a {{PRODUCT_NAME}} API key.

## 2. Check the service

Verify connectivity with the health endpoint:

```bash
curl https://api.irora.ai/{{PRODUCT_SLUG}}/v1/health
```

```json
{
  "status": "ok",
  "service": "{{PRODUCT_SLUG}}",
  "timestamp": "2026-03-03T12:00:00.000Z"
}
```

## 3. Make your first request

<CodeGroup>
```bash curl
curl -H "Authorization: Bearer YOUR_API_KEY" \
  https://api.irora.ai/{{PRODUCT_SLUG}}/v1/{{EXAMPLE_ENDPOINT}}
```

```javascript Node.js
const res = await fetch(
  "https://api.irora.ai/{{PRODUCT_SLUG}}/v1/{{EXAMPLE_ENDPOINT}}",
  { headers: { Authorization: "Bearer YOUR_API_KEY" } }
);
const data = await res.json();
console.log(data);
```

```python Python
import requests

res = requests.get(
    "https://api.irora.ai/{{PRODUCT_SLUG}}/v1/{{EXAMPLE_ENDPOINT}}",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
)
print(res.json())
```
</CodeGroup>
